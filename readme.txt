After installing the node (above version 12)

Command to setup node modules:(if package.json is available)
- npm i

Commands needed to install different packages:(if dont have package.json)
- npm init
- npm i fs	
- npm install --save-dev esm
- npm i puppeteer
- npm install -g lighthouse
- npm install exceljs


Command to run program: 
- node -r esm Lighthouse-main.js
